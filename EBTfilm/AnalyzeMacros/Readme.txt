- these macros create main histograms to analyze dose field distribution:
  doseHist.C - the main macro producing histos and saving them into ROOT file
  Portraited.h - calibration table
  
- Tif2Root.bat - helper batch file to run doseHist.C macro

===============================================================
Execute Tif2RootReg.bat to associate .tif file extension to execution 
of doseHist.C macro on double click.
