Процедура калибровки:

1. скопируйте файлы из папки CalibrEBT в папку с калибровочными файлами
2. перейдите в папку с калибровочными файлами (total commander)
3. запустите из коммандной строки root
4. следуйте инструкциям появляющимся на экране
